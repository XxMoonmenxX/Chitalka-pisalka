# Chitalka-pisalka
Программа для написания книг с удобным редактором
Разработано при помощи ИИ.

Скачать можно во вкладке Releases  или [ТЫК](https://github.com/XxMoonmenxX/Chitalka-pisalka/releases/download/v1.0/Pisalka.exe)


Читалка-писалка Pro — это профессиональный инструмент для писателей, позволяющий удобно организовывать процесс написания книг, рассказов и других литературных произведений. Программа поддерживает многодокументный режим, систему глав, настраиваемые темы оформления и экспорт в популярные форматы.

Основные возможности:

Создание и управление проектами книг

Иерархическая структура глав

Многодокументный интерфейс (одновременная работа с несколькими главами)

Настраиваемые темы оформления (5 встроенных тем)

Поддержка обложек книг

Vim-режим для продвинутых пользователей

Экспорт в FB2, PDF, TXT

Автосохранение и резервное копирование

Статистика по проекту в реальном времени

Установка и запуск
Системные требования
ОС: Windows 7/8/10/11 (64-bit)

Процессор: 1 ГГц или выше

ОЗУ: минимум 512 МБ (рекомендуется 1 ГБ)

Место на диске: 100 МБ для программы, дополнительно для проектов

Установка
Распакуйте архив ChitalkaPisalka_Pro.zip в любую удобную папку

Запустите файл ChitalkaPisalka_Pro.exe

При первом запуске программа создаст папку .chitalkapisalka в вашей домашней директории для хранения настроек и резервных копий

Первый запуск
При первом запуске вы увидите стартовую страницу с предложением:

Создать новый проект

Открыть существующий проект

Выбрать проект из списка недавних

Интерфейс программы
https://interface.png

Основное окно программы разделено на несколько областей:

1. Главное меню
Файл — создание, открытие, сохранение проектов, экспорт

Настройки — темы оформления, настройки редактора

Справка — информация о программе

2. Панель инструментов
Быстрый доступ к основным функциям:

📄 Новый проект

📂 Открыть проект

💾 Сохранить всё

➕ Добавить главу

🖼️ Обложка

⚙️ Настройки

3. Панель "Проект" (слева)
Дерево глав текущего проекта. Здесь можно:

Открывать главы двойным щелчком

Перетаскивать главы для изменения порядка

Вызывать контекстное меню (правый клик) для переименования, удаления, изменения статуса

4. Рабочая область (центр)
MDI-область (Multiple Document Interface), где открываются редакторы глав. Каждая глава открывается в отдельной вкладке/окне.

5. Панель "Статистика" (справа)
Отображает:

Предпросмотр обложки

Количество глав, слов, символов

Статусы глав (черновик, на ревью, готово)

Даты создания и изменения проекта

6. Панель "Поиск" (внизу)
Поиск по всем главам проекта.

7. Строка состояния
Информация о текущих действиях и статусе программы.

Работа с проектами
Создание нового проекта
Нажмите Файл → Новый проект (или кнопку на панели инструментов, или Ctrl+N)

В появившемся диалоге укажите:

Название проекта — обязательное поле

Автор — ваше имя (по умолчанию "Автор")

Папка для сохранения — можно оставить автоматическую или выбрать другую

Обложка — можно выбрать изображение сейчас или добавить позже

Нажмите Создать

Проект будет сохранен в указанной папке со следующей структурой:


Название_проекта/

├── project.json        # метаданные проекта

├── chapters/           # папка с текстами глав

│   ├── {id1}.txt

│   └── {id2}.txt

└── covers/             # папка с обложками (если есть)

  └── cover_{id}.jpg
    
Открытие проекта
Способы открытия:

Файл → Открыть проект... (Ctrl+O) — выбрать папку проекта

Из списка Недавние проекты в меню Файл

Со стартовой страницы

Сохранение
Автосохранение — включается в настройках (по умолчанию каждые 30 секунд)

Сохранить всё (Ctrl+Shift+S) — сохраняет все открытые главы и проект

При закрытии программы будет предложено сохранить изменения

Резервное копирование
При включенной опции (в настройках) программа создает ZIP-архивы проектов в папке:


C:\Users\[Имя]\.chitalkapisalka\backups\
Хранится до 10 последних копий (настраивается).

Управление главами
Добавление главы
Нажмите кнопку Добавить главу на панели инструментов

Введите название главы (по умолчанию "Глава N")

Нажмите Создать

Глава появится в дереве проекта.

Открытие главы
Двойной щелчок по главе в дереве проекта

Глава откроется в новой вкладке в центральной области

Редактирование главы
В открытом редакторе главы доступны:

Поле названия — можно изменить название главы

Выпадающий список статуса — черновик / на ревью / готово

Статистика — количество слов и символов в текущей главе

Текстовый редактор — основное поле для ввода текста

Переименование главы
Правый клик по главе в дереве проекта

Выберите Переименовать

Введите новое название

Нажмите ОК

Или измените название в поле редактора главы.

Изменение статуса главы
Через контекстное меню в дереве проекта

Через выпадающий список в редакторе главы

Статусы:

📝 Черновик — работа в процессе

🔍 На ревью — требует проверки

✅ Готово — завершенная глава

Изменение порядка глав
Просто перетащите главу мышкой в нужное место в дереве проекта.

Удаление главы
Правый клик по главе

Выберите Удалить

Подтвердите удаление

Внимание: удаление главы необратимо!

Работа с обложками
Добавление обложки при создании проекта
В диалоге создания проекта нажмите Выбрать... рядом с полем "Обложка" и укажите файл изображения.

Добавление/изменение обложки в существующем проекте
Способы:

Файл → Обложка → Добавить/Изменить обложку

Кнопка Обложка на панели инструментов

Через диалог управления обложкой в статистике проекта

Поддерживаемые форматы: PNG, JPG/JPEG, BMP, GIF

Просмотр обложки
Файл → Обложка → Просмотреть обложку

Обложка также отображается в панели статистики

Удаление обложки
Файл → Обложка → Удалить обложку

Особенности
Обложка автоматически встраивается в экспортируемые FB2 и PDF файлы

При экспорте в TXT указывается только имя файла обложки

Обложки хранятся в папке проекта (covers/)

Редактор текста
Основные возможности
Подсветка текущей строки

Настраиваемый шрифт и размер

Перенос слов (включается/выключается в настройках)

Поддержка Vim-режима

Vim-режим
При включенном Vim-режиме доступны два режима:

Режим вставки (-- ВСТАВКА --)

Обычный ввод текста

Выход в нормальный режим: Esc

Нормальный режим (-- НОРМАЛЬНЫЙ --)

h/j/k/l — перемещение курсора

x — удалить символ под курсором

i — перейти в режим вставки

Ctrl+U — отмена действия (undo)

Статистика в редакторе
В верхней панели редактора отображается количество слов и символов в текущей главе (обновляется в реальном времени).

Настройки
Открытие настроек
Настройки → Настройки... (Ctrl+,)

Кнопка ⚙️ на панели инструментов

Вкладка "Общие"
Автосохранение

Включить/выключить автосохранение

Интервал в секундах (10-300)

Резервное копирование

Включить создание резервных копий

Количество хранимых копий (1-50)

Недавние проекты

Максимальное количество проектов в списке (3-30)

Вкладка "Оформление"
Темы оформления

Тёмная — классическая тёмная тема

Светлая — светлая тема

Сепия — тёплая тема для чтения

Дракула — тёмная тема с фиолетовыми акцентами

Monokai — яркая тема для программистов

При выборе темы можно сразу увидеть предпросмотр в диалоге настроек.

Вкладка "Редактор"
Шрифт

Семейство шрифта (моноширинные шрифты рекомендуется)

Размер шрифта (8-72)

Режимы

Vim-режим — включение/выключение

Перенос слов — включение/выключение

Табуляция

Размер табуляции (2-8 пробелов)

Экспорт книг
Экспорт в FB2 (FictionBook)
Самый функциональный формат, поддерживающий:

Обложку

Метаданные (автор, название)

Аннотацию

Структуру глав

Процесс экспорта:

Файл → Экспорт → FB2

Укажите имя файла и место сохранения

Нажмите Сохранить

После завершения программа предложит открыть созданный файл.

Экспорт в PDF
Поддерживает:

Обложку (если есть) на отдельной странице

Заголовки глав

Разрывы страниц между главами

Процесс экспорта:

Файл → Экспорт → PDF

Укажите имя файла и место сохранения

Нажмите Сохранить

Экспорт в TXT
Простой текстовый формат:

Заголовок с названием и автором

Информация о файле обложки (если есть)

Главы с заголовками

Горячие клавиши

Ctrl+Shift+S	Сохранить всё

Решение проблем

Программа не запускается 

Убедитесь, что ваш компьютер соответствует системным требованиям 

Попробуйте запустить от имени администратора 

Проверьте, не блокирует ли антивирус программу 

Переустановите программу 

Не сохраняются настройки 

Проверьте права доступа к папке C:\Users\[Имя]\.chitalkapisalka\

Удалите файл config.json в этой папке (настройки сбросятся)

Не открывается проект

Убедитесь, что папка проекта содержит файл project.json

Проверьте, не поврежден ли файл (откройте в текстовом редакторе)

Восстановите проект из резервной копии

Не экспортируется FB2

Проверьте, что в названиях глав нет недопустимых символов

Убедитесь, что файл не заблокирован другой программой

Попробуйте сохранить в другую папку

Обложка не отображается

Проверьте формат изображения (поддерживаются PNG, JPG, BMP, GIF)

Убедитесь, что файл не поврежден

Переустановите обложку

Программа зависает

Подождите несколько секунд (возможно, идет автосохранение)

Нажмите Ctrl+Shift+Esc и закройте процесс в диспетчере задач

Проверьте, достаточно ли свободной памяти

Где найти резервные копии?

Резервные копии хранятся в папке:

C:\Users\[Имя]\.chitalkapisalka\backups\
Как перенести проекты на другой компьютер?
Скопируйте папку проекта (содержит все главы и обложку)

На новом компьютере откройте проект через Файл → Открыть проект...

Программа автоматически добавит его в список недавних
